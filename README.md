Overview:
The Soccer xG Calculator is a program designed to analyze user-provided data on soccer shot events and calculate the expected goals (xG) value using a sophisticated xG model. This tool is useful for coaches, analysts, and soccer enthusiasts who want to quantify the quality of scoring chances in a match.


Features:
User-friendly input for shot location and other relevant metrics,
Calculation of xG based on an established model,
Clear output of xG value for each shot.


Model Details:
The xG model used in this program is based on historical shot data and various factors such as shot location, type, and context from the English Premier League. For more detailed information on the methodology behind the model, please refer to the research papers and articles on expected goals.
